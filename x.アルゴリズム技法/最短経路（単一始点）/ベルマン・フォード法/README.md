# ベルマン・フォード法

## 特徴
- **重み付けグラフ**における、**単一始点最短経路** 問題で用いる
- **負の重みを許容** する
- **ダイクストラ法** がより高速

## 手法
- 全ての辺に注目し、先の頂点を緩和できそうなら緩和する
- 以上を `N - 1` 回繰り返す（`N` は頂点数）
- 以上を再度繰り返し、値が更に緩和できれば**負閉路**が存在する

## 原理
- 始点から１辺ずつ確定距離を算出できる
  - その他の辺は意味のある犠牲となる
- これを `N - 1` 回行うことで終点まで確実に到達する

## 計算量
- `O(V^2 +E)`
